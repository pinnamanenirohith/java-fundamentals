## Overview

This repository contains **structured Java programming practice** based on **Wipro TalentNext (WTN)** learning.

The repository includes topic-wise Java concepts, hands-on programs, coding exercises, and object-oriented programming implementations for **placement preparation** and software development fundamentals.

---

## Topics Covered

| # | Topic | Description |
|---|-------|-------------|
| 01 | Java Basics | JDK, JRE, JVM, Platform Independence |
| 02 | Data Types | Primitive types, Variables, Type Conversion |
| 03 | Operators | Arithmetic, Relational, Logical, Bitwise |
| 04 | Conditional Statements | if, if-else, switch case |
| 05 | Loops | for, while, do-while, nested loops |
| 06 | Arrays | 1D, 2D Arrays, Searching, Sorting |
| 07 | Strings | String methods, StringBuilder, StringBuffer |
| 08 | Methods | User-defined, Overloading, Static methods |
| 09 | Wrapper Classes | Autoboxing, Unboxing, Character methods |
| 10 | OOP Concepts | Encapsulation, Inheritance, Polymorphism, Abstraction |
