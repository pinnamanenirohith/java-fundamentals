# 02 - Data Types

## Overview

This section covers Java **Data Types**, including primitive types, variables, type conversion, and a brief introduction to Wrapper Classes. Understanding data types is fundamental for writing correct and efficient Java programs.

## Key Concepts

**Primitive Data Types in Java:**

| Type | Size | Default | Range |
|------|------|---------|-------|
| byte | 1 byte | 0 | -128 to 127 |
| short | 2 bytes | 0 | -32,768 to 32,767 |
| int | 4 bytes | 0 | -2^31 to 2^31-1 |
| long | 8 bytes | 0L | -2^63 to 2^63-1 |
| float | 4 bytes | 0.0f | ~3.4e-038 to ~3.4e+038 |
| double | 8 bytes | 0.0d | ~1.7e-308 to ~1.7e+308 |
| char | 2 bytes | '\u0000' | 0 to 65,535 |
| boolean | 1 bit | false | true or false |

---

## Learning Objectives

After completing this module, you will be able to:
- Declare and use all 8 primitive data types in Java
- Understand variable scope and initialization rules
- Perform type casting (widening and narrowing conversions)
- Use Wrapper Classes for primitive types
