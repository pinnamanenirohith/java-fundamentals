# 06 - Arrays

## Overview

This section covers **Arrays** in Java — a data structure that stores a fixed-size sequential collection of elements of the same type. Arrays are essential for storing and manipulating collections of data.

## Key Concepts

**Array Declaration and Initialization:**

```java
// Declaration
int[] arr;

// Initialization
int[] arr = new int[5];

// Declaration + Initialization
int[] arr = {10, 20, 30, 40, 50};

// 2D Array
int[][] matrix = new int[3][3];
int[][] matrix = {{1,2,3}, {4,5,6}, {7,8,9}};
```

**Important Properties:**
- Arrays in Java are zero-indexed (start from index 0)
- `arr.length` gives the number of elements
- Arrays have a fixed size once created
- Default values: `0` for int, `null` for objects, `false` for boolean
