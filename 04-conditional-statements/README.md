# 04 - Conditional Statements

## Overview

This section covers **Conditional Statements** in Java — control flow mechanisms that allow programs to make decisions and execute different code blocks based on conditions.

---

## Topics Covered

| # | Topic | Description |
|---|-------|-------------|
| 1 | if statement | Execute code block if condition is true |
| 2 | if-else statement | Choose between two code blocks |
| 3 | nested if | if statements inside other if blocks |
| 4 | else-if ladder | Multiple conditions checked in sequence |
| 5 | switch case | Multi-way branch based on a variable value |

---

## Key Concepts

**Syntax Overview:**

```java
// if statement
if (condition) {
    // code
    }

    // if-else statement
    if (condition) {
        // code if true
        } else {
            // code if false
            }

            // else-if ladder
            if (condition1) {
                // code
                } else if (condition2) {
                    // code
                    } else {
                        // default code
                        }

                        // switch statement
                        switch (variable) {
                            case value1:
                                    // code
                                            break;
                                                case value2:
                                                        // code
                                                                break;
                                                                    default:
                                                                            // default code
                                                                            }
                                                                            ```
