## Key Concepts

**JDK vs JRE vs JVM:**

| Component | Full Form | Role |
|-----------|-----------|------|
| JVM | Java Virtual Machine | Executes Java bytecode on any platform |
| JRE | Java Runtime Environment | JVM + libraries needed to run Java programs |
| JDK | Java Development Kit | JRE + development tools (compiler, debugger) |

**Java Compilation Flow:**

```
Source Code (.java)
       |
  javac compiler
       |
  Bytecode (.class)
       |
  JVM (Java Virtual Machine)
       |
  Platform Output
```

**Java Features:**
- Simple and Object-Oriented
- - Platform Independent (WORA)
  - - Robust and Secure
    - - Multithreaded
      - - High Performance
        - - Distributed and Dynamic
