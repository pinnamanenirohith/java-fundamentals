# 05 - Loops

## Overview

This section covers **Loops** in Java — control flow structures that allow repetitive execution of code blocks. Loops are fundamental to writing efficient programs that handle repetitive tasks.

---

## Topics Covered

| # | Topic | Description |
|---|-------|-------------|
| 1 | for loop | Definite loop with initialization, condition, and update |
| 2 | while loop | Loop that executes while condition is true |
| 3 | do-while loop | Loop that executes at least once, checks condition after |
| 4 | nested loops | Loops inside other loops for multi-dimensional iteration |
| 5 | break and continue | Loop control statements for early exit and skipping |

---

## Key Concepts

**Loop Syntax:**

```java
// for loop
for (int i = 0; i < n; i++) {
    // code
    }

    // while loop
    while (condition) {
        // code
        }

        // do-while loop
        do {
            // code
            } while (condition);

            // Enhanced for loop (for-each)
            for (dataType variable : array) {
                // code
                }
                ```

                **break vs continue:**
                - `break` — Exits the loop immediately
                - `continue` — Skips current iteration and moves to next
